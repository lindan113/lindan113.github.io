Java Note 2

Note of Core Java :)

Classes

  • Classes are the building blocks with which all Java applications and applets are built.
  • Everything in a Java program must be inside a class.
  • You need to make the file name for the source code the same as the name of the public class, with the extension .java appended. E.g. we have a public class HelloWorld, we must store this code in a file called HelloWorld.java.

When you run a compiled program, the Java virtual machine always starts execution with the code in the main method in the class you indicate.
(The term “method” is Java-speak for a function.)
According to the Java Language Specification, the main method must be declared public . The main method in Java is always static.

  • In Java all functions are methods of some class.

  • Java uses the general syntax
    object.method ( parameters )

    as its equivalent of a function call.
    E.g.
    System.out.println("We will not use 'Hello, World!'");

  • Call the println method with no parameters just prints a blank line
    System.out.println();

  • System.out also has a print method that doesn’t add a newline character to the output.
    System.out.print("Hello") prints Hello without a newline.

Caution:

/* */ comments do not nest in Java.

Math class

The Math class contains some mathematical functions that you may often
need.

  • The Java programming language has no operator for raising a quantity to a power: You must use the pow method in the Math class. The statement
    double y = Math.pow(x, a);

sets y to be x raised to the power a.

  • The Math class supplies the usual trigonometric functions, e.g. Math.sin, Math.log
    If you want to round a floating-point number to the nearest integer (which in most cases is the more useful operation), use the Math.round method:
    double x = 9.997;
    int nx = (int) Math.round(x);
    Now the variable nx has the value 10 .

  • You can avoid the Math prefix for the mathematical methods and constants by adding the following line to the top of your source file:
    import static java.lang.Math.*;

String

  • Unlike C++, Java contains a predefined class called String( Caution: the first letter is uppercase)

  • The C++ string class does overload the == operator to test for equality of the string contents. But in Java, to test whether two strings are equal, use the equals method. The expression
    s.equals(t)
    returns true if the strings s and t are equal, false otherwise. Note that s and t can be string variables or string constants. For example, the expression
    “Hello”.equals(greeting)

  • The empty string “” is a string of length 0 .
  • If no object is currently associated with the String variable, it holds a special value, called null ,
  • Sometimes, you need to test that a string is neither null nor empty. Then use the condition
    if (str != null && str.length() != 0)

Java Note 1

Note of Core Java :)

Data Type

There are eight primitive types in Java. Four of them are integer types; two are floating- point number types; one is the character type char , used for code units in the Unicode encoding scheme。


Note: Big Numbers

Java has an arbitrary-precision arithmetic package. However, “big numbers,” as they are called, are Java objects and not a new Java type. We will see how to use them later in this chapter.

The boolean type has two values, false and true .In C++, The value 0 is equivalent to the bool value false , and a non-zero value is equivalent to true . This is not the case in Java.

Constant

In C++, we use const to denote a constant. In Java, you use the keyword final. The keyword final indicates that you can assign to the variable once, and then its value is set once and for all. It is popular to name constants in all uppercase.

Operators

“>>” and “<<” operators shift a bit pattern to the right or left. a >>> operator fills the top bits with zero, unlike >> which extends the sign bit into the top bits. There is no <<< operator.
