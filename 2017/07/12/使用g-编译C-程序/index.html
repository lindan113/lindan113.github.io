使用g++编译C++程序

使用g++编译一个C++程序, 输出Hello World

  1. 打开Linux系统，在某个文件夹下，写一个c/cpp文件,输出”Hello world!”；
  2. 打开Linux终端（快捷键是Ctrl+Alt+T）；
  3. 在终端，通过cd命令进入cpp文件所在目录；
  4. 接着输入 g++ helloworld.cpp -o helloworld ，其中helloworld是文件名，在当前目录下得到编译的文件；
  5. 将编译文件拖入终端，按回车执行程序，或者输入./ helloworld，可以看到终端输出”Hello world!”；
